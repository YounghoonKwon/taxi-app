  - [ ] 예외: 주소가 없거나 주소의 허용된 문자열 길이 1-100자를 벗어난 경우 400

- [ ] 배차 수락

  - [ ] 예외: 기사가 아닌 유저가 배차 요청한 경우 403

  - [ ] 예외: 이미 다른 기사에 의해 배차 요청이 수락된 경우 409
